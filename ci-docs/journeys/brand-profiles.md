# Create consistent branding with brand profiles

> [!NOTE]
> Unless a style is explicitly set in the element properties, the default style is replaced with the email theme style.

Brand profiles let you create consistently branded content efficiently, even if your organization has multiple brands. You can create multiple brand profiles for your organization or profiles for each of your business units. You can also define default senders for emails and set default social links. You can even extend your brand profile entity with custom values that you can use when creating content. This article describes how to create a brand profile.

## Create a brand profile

Brand profile settings are available in the **Settings** area in the **Customer engagement** section.

:::image type="content" source="media/brand-profiles-settings.png" alt-text="Screenshot of the Brand profiles area in Settings." lightbox="media/brand-profiles-settings.png":::

To create a brand profile, select **+New** in the top navigation and define the profile's name. Optionally, you can also describe it. Make sure to save it when you're done.

:::image type="content" source="media/brand-profile-new.png" alt-text="Screenshot of the New button used to create a brand profile." lightbox="media/brand-profile-new.png":::

### Add senders to a brand profile

The **Senders** tab in the brand profile lets you define senders associated with your profile. To add senders, select the **Senders** tab then select **+New Sender**. You can add as many senders as your organization needs.

When adding a sender, specify the "From email," "From name," and "Reply-to email." You can also choose a default sender by setting the "Default" value to **Yes**. Make sure to select **Save and Close** to save the new sender.

:::image type="content" source="media/brand-profiles-create-sender.png" alt-text="Screenshot of the dialog for creating a new sender in a brand profile." lightbox="media/brand-profiles-create-sender.png":::

### Define social links for a brand profile

To define social links for your brand profile, open the **Social links** tab and add the links you need.

:::image type="content" source="media/brand-profiles-social-links.png" alt-text="Screenshot of the Social links tab in a brand profile." lightbox="media/brand-profiles-social-links.png":::

After you've added your links, select **Save**.

### Define a brand profile theme

Open the **Theme** tab in the brand profile and add the theme in accordance with your brand guidelines. You can specify any custom fonts in the theme section (the theme section in the email editor also contains these options). Once your theme is added, select **Save** to automatically apply your theme to all draft content that includes a brand profile.

:::image type="content" source="media/brand-profiles-themes.png" alt-text="Screenshot of the Theme tab in a brand profile." lightbox="media/brand-profiles-themes.png":::

And that's it! Your profile is created and is ready to use in your emails. Learn more: [Use brand profiles in email](brand-profiles-email.md).

> [!IMPORTANT]
> The brand profile table is editable, so you can add new fields to it. For example, you could add fields to store URL links for more social platforms.
