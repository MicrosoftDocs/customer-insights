# Use the versions page to check and update the current solution version

The versions page shows you the current version of the solutions on your environment and the status of those solutions relative to what's been released.

If new solution updates are available, select **Manage+Update** to launch the installation management page. Find the environment you'd like to update, select **Check Versions**, and then choose **Update**.

[!NOTE]
> You may see an additional area with version details for the outbound marketing module, which reached its end of life on June 30, 2025. You can ignore this information; it is temporary and will be removed in a future release.
